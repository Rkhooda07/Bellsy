Prefix any terminal command with bellsy -- to start monitoring.

A native OS notification alerts you the moment it finishes, fails, or needs an approval.

Install in seconds

Requires Node.js 18+

npm install -g bellsy

Search for "Bellsy" in the VS Code Marketplace or run:

code --install-extension bellsy.agent-monitor

Configuration

Setting Default Description
bellsy.soundMode "subtle" Sound notification style: "none", "subtle", or "loud".
bellsy.httpPort 9009 Local port for the HTTP trigger server.
bellsy.autoWrap false Automatically monitor all long-running (>1m) shell tasks.
