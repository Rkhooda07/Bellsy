How it works

01

Wrap your command

Run bellsy-run gemini or bellsy-run claude — Bellsy starts a background server and wraps your process.

02

Bellsy watches

Monitors stdout and stderr in real time. Detects completions, errors, and approval-needed states automatically.

03

You get notified

Native macOS notification, VS Code popup, or a sound cue — whichever fits your flow.

Everything you need

Smart Notifications

Native macOS alerts with click-through to bring VS Code into focus.

Sound Modes

Focus (subtle) and Vibe (playful) — configurable volume and mode.

Approval Reminders

Periodic prompts when an agent is waiting on you so nothing stalls.

Agent Integrations

Works with Claude, Gemini, Codex, and any custom script out of the box.

Local HTTP Endpoint

Post JSON to 127.0.0.1:9001/event from any process and get notified.

Fully Configurable

Port, volume, reminder interval — all via VS Code settings.json.
